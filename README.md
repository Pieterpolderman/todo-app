# To-Do List App 📝

This is a simple To-Do List app built with **Next.js** and **React**, styled using **Tailwind CSS**.

## 🚀 Features
- Add and remove tasks
- Simple UI with Tailwind CSS
- Deployed with Vercel (optional)

## 🛠 Technologies Used
- **Next.js**
- **React**
- **Tailwind CSS**
